## 核心项目集 (Portfolio)

### 📁 项目一：大 DAU 射击手游长线运营与商业化重构
**研究对象**：《逆战：未来》（千万级新增大盘产品）
**项目背景**：针对游戏公测初期面临的口碑分化、内容消耗过快等痛点，独立输出的深度长线运营破局方案。

**核心亮点：**
* **数据驱动洞察**：抓取并清洗 200+ 真实玩家高频评论，通过 NLP 词频与情感倾向分析（负面情绪占比定位），精准拆解“付费体验差”、“内容空洞”等 6 大核心痛点。
* **分阶段增长模型**：量化拆解 3个月（留存修复）、6个月（用户召回）、12个月（破圈拉新）的阶梯式运营 KPI。
* **社群与内容裂变**：结合短视频时代特征，设计“碎片化叙事”与 UGC 达人裂变机制，将剧情转化为玩家的社交货币。

### 项目二：基于 LLM 的游戏运营数据智能诊断引擎 (AI Agent)
**项目背景**：传统游戏运营每日拉取多渠道数据、计算同环比、撰写日报极其耗时。为实现业务提效，从零搭建基于 Dify 工作流的自动化引擎。

**核心亮点：**
* **解决大模型计算幻觉**：前置 Python 清洗脚本处理 90 行三端（iOS、安卓、官网）复杂 CSV 脏数据，保障 DAU、ARPU、流水等财务级数据 100% 精准。
* **智能异动告警**：设定业务阈值（如核心指标环比下跌超 10%），触发 🚨 红色预警，辅助运营快速定位异常点（如节假日回落或服务器宕机）。
* **一键生成结构化简报**：设计高阶 Prompt，秒级输出企业级 Markdown 格式数据看板与次日运营动作建议。
